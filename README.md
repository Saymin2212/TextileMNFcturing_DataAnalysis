# Textile Manufacturing
Textile Manufacturing Data Analysis Project
This repository contains a comprehensive end-to-end data analysis project for the Textile Manufacturing industry, showcasing business performance, operational efficiency, quality metrics, and workforce insights using multiple data analytics tools.

📊 Project Overview:

1. The project focuses on analyzing manufacturing data to evaluate production performance, rejection trends, delivery efficiency, departmental costs, customer deliveries, and employee productivity.
2. It demonstrates how insights can be derived from raw data and translated into interactive dashboards and executive-level presentations.

🧵 Industry Focus

Textile Manufacturing, covering key departments such as:
1. Knitwear
2. Footwear
3. Woven Labels
4. Printed Fabric

🛠 Tools & Technologies Used

1. SQL – Data extraction, cleaning, joins, aggregations, and KPI calculations
2. Power BI – Interactive dashboards with slicers, drill-downs, and insights
3. Tableau – Visual storytelling and comparative analysis
4. Microsoft Excel – Exploratory analysis and pivot-based summaries
5. PowerPoint (PPT) – Final business presentation with insights and recommendations

🎯 Project Objective
The objective of this project is to demonstrate a complete manufacturing data analytics workflow, from raw data analysis using SQL to interactive dashboards and executive-level reporting.
